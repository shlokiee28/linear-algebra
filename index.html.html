Chapter 01

Operations of Vectors

Theory

A vector is a mathematical object with both magnitude and direction. Unlike scalars (plain numbers), vectors live in multi-dimensional space and encode positional relationships. In ℝⁿ, a vector is an ordered list of n real numbers arranged as a column.

Vectors are the atoms of linear algebra — everything else (matrices, transformations) is built on top of them. Geometrically, you can think of a vector as an arrow rooted at the origin, pointing to a coordinate in space. Operations on vectors correspond to geometric moves: addition shifts the arrow's tip, scalar multiplication stretches or compresses it, and negation reverses its direction.

Vectors obey specific algebraic laws: commutativity (u + v = v + u), associativity ((u + v) + w = u + (v + w)), and the existence of a zero vector that acts as an additive identity. These properties form the foundation of a vector space.

  • Vector Representation In ℝⁿ, a vector v is written as a column: v = (v₁, v₂, …, vₙ)ᵀ. The superscript T denotes transpose — turning a row into a column. In ℝ², v = (x, y)ᵀ gives a point in the plane. In ℝ³, v = (x, y, z)ᵀ gives a point in space.
  • Vector Addition Component-wise sum: U + V = (U₁+V₁, U₂+V₂, …, Uₙ+Vₙ)ᵀ. Geometrically, place the tail of V at the tip of U — the result is the vector pointing from the origin to the new tip. This is the parallelogram law.
  • Vector Subtraction Equivalent to adding the negative: U − V = U + (−V) = (U₁−V₁, …, Uₙ−Vₙ). The result vector points from V's tip to U's tip. Used to find displacement vectors between two points in space.
  • Scalar Multiplication c · V = (cV₁, cV₂, …, cVₙ)ᵀ. Scales the vector's magnitude by |c| and flips direction when c < 0. The direction is preserved for c > 0. When c = 0, the result is the zero vector.
  • Linear Combination A sum of the form c₁v₁ + c₂v₂ + … + cₖvₖ where cᵢ are scalars and vᵢ are vectors. Any vector in the span of {v₁, …, vₖ} can be written as a linear combination. This idea is central to understanding vector spaces and bases.
Example — Addition, Subtraction & Scalar Cases
// Vector Addition
U = (1, 2, 3)ᵀ   V = (4, 5, 6)ᵀ

U + V = (1+4, 2+5, 3+6)ᵀ = (5, 7, 9)ᵀ

// Vector Subtraction
U - V = (1−4, 2−5, 3−6)ᵀ = (−3, −3, −3)ᵀ

// Scalar Multiplication Cases
c = 2    →  Doubles the vector magnitude
c = 0.5  →  Halves the vector magnitude
c = 1    →  No change
c = 0    →  Zero vector
c = -1   →  Reverses direction

// Linear Combination
2·(1,0,0)ᵀ + 3·(0,1,0)ᵀ + 5·(0,0,1)ᵀ = (2, 3, 5)ᵀ
Chapter 02

Dot Product & Norms

Theory

The dot product (inner product) collapses two vectors into a single scalar, encoding the geometric relationship between them. It is the cornerstone of concepts like orthogonality, projection, and cosine similarity used throughout machine learning and physics.

The norm of a vector is its length — derived from the dot product of the vector with itself. When two vectors have a dot product of zero, they are orthogonal (perpendicular), meaning they share no common directional component. This is the algebraic definition of "right angle" in any dimension.

The dot product has a beautiful geometric interpretation: U · V = |U||V|cos(θ). It is positive when vectors point in similar directions, negative when they point in opposite directions, and exactly zero when they are perpendicular. This makes it a measure of "alignment" between vectors.

  • Dot Product (Algebraic) U · V = U₁V₁ + U₂V₂ + ⋯ + UₙVₙ = Σ UᵢVᵢ. The result is always a scalar. The dot product is commutative (U·V = V·U) and distributive over addition.
  • Dot Product (Geometric) U · V = |U||V|cos(θ), where θ is the angle between the two vectors. This formula connects the algebraic computation to the geometric angle, and is used to find angles between vectors in any dimension.
  • Euclidean Norm (L2) ‖V‖ = √(V·V) = √(V₁² + V₂² + ⋯ + Vₙ²). This is the vector's length in n-dimensional space, a direct generalisation of the Pythagorean theorem. Also called the L2 norm or 2-norm.
  • Unit Vector A vector normalized to length 1: V̂ = V / ‖V‖. Unit vectors encode direction only — all magnitude information is removed. The standard basis vectors î = (1,0,0), ĵ = (0,1,0), k̂ = (0,0,1) are all unit vectors.
  • Orthogonality U ⊥ V ⟺ U · V = 0. Two vectors are orthogonal if and only if their dot product is zero. A set of vectors that are all mutually orthogonal and unit length is called an orthonormal set — the basis of choice in most applications.
  • Cauchy-Schwarz Inequality |U · V| ≤ ‖U‖ · ‖V‖. The absolute value of the dot product is always at most the product of the norms. Equality holds when U and V are parallel (collinear). This is one of the most important inequalities in all of mathematics.
Example — Dot Product, Norm & Angle
// Dot Product Calculation
U = (3, 4)ᵀ     V = (1, 0)ᵀ

U · V = 3×1 + 4×0 = 3

// Norms
‖U‖ = √(3² + 4²) = √(9 + 16) = 5
‖V‖ = √(1² + 0²) = 1

// Angle between U and V
cos(θ) = U·V / (‖U‖ × ‖V‖) = 3 / (5 × 1) = 0.6
θ      = arccos(0.6) ≈ 53.13°

// Unit Vector of U
Û = U / ‖U‖ = (3, 4)ᵀ / 5 = (0.6, 0.8)ᵀ   ‖Û‖ = 1 ✓

// Orthogonal check
A = (1, 0)ᵀ   B = (0, 1)ᵀ   →   A·B = 0  → Orthogonal ✓

// Cauchy-Schwarz verification
|U·V| = 3  ≤  ‖U‖·‖V‖ = 5×1 = 5   ✓
Chapter 03

MATRICES

Theory

A matrix is a rectangular grid of numbers with m rows and n columns, denoted as an m×n matrix. Matrices are the primary data structure for encoding linear transformations, systems of equations, and multi-dimensional relationships. Every matrix implicitly defines a function from ℝⁿ to ℝᵐ.

Matrix multiplication is not just number-crunching — it is function composition. The product AB means "apply transformation B first, then A." This non-commutativity (AB ≠ BA in general) reflects the order-dependence of sequential transformations. The determinant of a square matrix measures how much the matrix scales area (or volume) — a zero determinant means the matrix collapses space and is non-invertible.

Special matrix types appear everywhere: the identity matrix I acts as the multiplicative identity (AI = IA = A), diagonal matrices scale along coordinate axes, and orthogonal matrices (where Aᵀ = A⁻¹) represent pure rotations and reflections.

  • Matrix Shape & Indexing An m×n matrix has m rows and n columns. The (i,j) entry is in row i, column j. Shape determines what operations are valid: multiplication requires inner dimensions to match — an m×k matrix times a k×n matrix gives an m×n result.
  • Matrix Addition & Scalar Multiplication Matrices of the same shape can be added element-wise: (A+B)ᵢⱼ = Aᵢⱼ + Bᵢⱼ. Scalar multiplication scales every entry: (cA)ᵢⱼ = c·Aᵢⱼ. These operations make the set of m×n matrices a vector space.
  • Matrix Multiplication (AB)ᵢⱼ = Σₖ AᵢₖBₖⱼ. Each output entry is the dot product of a row from A with a column from B. Requires A to be m×k and B to be k×n, yielding an m×n result. Not commutative: AB ≠ BA in general.
  • Transpose Aᵀ flips A along its diagonal: (Aᵀ)ᵢⱼ = Aⱼᵢ. An m×n matrix transposed becomes n×m. Key identities: (AB)ᵀ = BᵀAᵀ and (Aᵀ)ᵀ = A. Symmetric matrices satisfy A = Aᵀ.
  • Determinant det(A) = ad − bc for a 2×2 matrix [[a,b],[c,d]]. For larger matrices, computed via cofactor expansion. Encodes the signed area (or volume) scaling factor of the transformation. det(A) = 0 means the matrix is singular (not invertible). det(AB) = det(A)·det(B).
  • Inverse Matrix A⁻¹ exists only if det(A) ≠ 0, and satisfies AA⁻¹ = A⁻¹A = I. For 2×2: A⁻¹ = (1/det(A)) · [[d,−b],[−c,a]]. The inverse "undoes" the transformation — if A maps v to w, then A⁻¹ maps w back to v.
Example — Matrix Multiplication, Determinant & Inverse
// 2×2 Matrix Multiplication
A = [[1, 2],    B = [[5, 6],
     [3, 4]]         [7, 8]]

AB = [[1×5+2×7,  1×6+2×8],
      [3×5+4×7,  3×6+4×8]]

   = [[19, 22],
      [43, 50]]

// Note: AB ≠ BA (non-commutative!)
BA = [[5×1+6×3,  5×2+6×4],   = [[23, 34],
      [7×1+8×3,  7×2+8×4]]        [31, 46]]

// Determinant of A
det(A) = (1×4) - (2×3) = 4 - 6 = -2

// Since det(A) ≠ 0, A is invertible
A⁻¹ = (1/-2) × [[ 4, -2],
                   [-3,  1]]

    = [[-2,   1  ],
       [ 1.5, -0.5]]

// Verify: A × A⁻¹ = I
A × A⁻¹ = [[1, 0],
            [0, 1]]  ✓
Chapter 04

Eigenvalues & Eigenvectors

Theory

An eigenvector of a matrix A is a special non-zero vector that, when multiplied by A, only gets scaled — it doesn't change direction. The corresponding scale factor is the eigenvalue. In other words: Av = λv, where v is the eigenvector and λ is the eigenvalue.

Eigenvectors reveal the "axes of action" of a transformation. They are foundational to PCA (Principal Component Analysis), Google's PageRank algorithm, quantum mechanics, and stability analysis of differential equations. Every real symmetric matrix has real eigenvalues and orthogonal eigenvectors — a deeply beautiful result called the Spectral Theorem.

The sum of all eigenvalues equals the trace (sum of diagonal entries) of the matrix, and their product equals the determinant. These relationships connect the eigenvalue structure directly to key matrix properties.

  • Eigenequation Av = λv means applying matrix A to vector v yields the same result as scaling v by scalar λ. The vector v is not rotated — only stretched or compressed. Rearranged: (A − λI)v = 0, which has non-trivial solutions only when det(A − λI) = 0.
  • Characteristic Polynomial Eigenvalues are roots of det(A − λI) = 0. This polynomial of degree n has exactly n roots (counting multiplicity, over ℂ), giving all eigenvalues of an n×n matrix. For a 2×2 matrix, this is always a quadratic equation.
  • Eigenspace For each eigenvalue λ, the set of all eigenvectors (plus the zero vector) forms the eigenspace: null(A − λI). This is a subspace of ℝⁿ. The dimension of this eigenspace is called the geometric multiplicity of λ.
  • Geometric Meaning of λ λ > 1: stretches the eigenvector. 0 < λ < 1: compresses it. λ = 1: eigenvector is unchanged. λ < 0: flips direction and scales. λ = 0: eigenvector is mapped to zero (lies in the null space of A).
  • Trace and Determinant Relations For an n×n matrix: sum of eigenvalues = tr(A) = Σ Aᵢᵢ (sum of diagonals). Product of eigenvalues = det(A). These are invariant shortcuts to check eigenvalue calculations without solving the full polynomial.
Example — Finding Eigenvalues & Eigenvectors
// Given matrix A
A = [[2, 1],
     [0, 3]]

// Step 1: Characteristic polynomial det(A - λI) = 0
det([[2-λ,  1 ],
     [ 0,  3-λ]]) = 0

(2-λ)(3-λ) - 0 = 0
λ² - 5λ + 6 = 0
(λ - 2)(λ - 3) = 0

// Eigenvalues
λ₁ = 2   λ₂ = 3

// Verify: trace = 2+3 = 5 = λ₁+λ₂ ✓   det = 6 = λ₁·λ₂ ✓

// Step 2: Eigenvector for λ₁ = 2
(A - 2I)v = [[0, 1],
             [0, 1]]v = 0   →   v₁ = (1, 0)ᵀ

// Step 3: Eigenvector for λ₂ = 3
(A - 3I)v = [[-1, 1],
             [ 0, 0]]v = 0  →   v₂ = (1, 1)ᵀ

// Verify: A·v₂ = λ₂·v₂
[[2,1],[0,3]] × (1,1)ᵀ = (3,3)ᵀ = 3 × (1,1)ᵀ  ✓
Chapter 05

LINEAR TRANSFORMATIONS

Theory

A linear transformation (or linear map) is a function T: ℝⁿ → ℝᵐ that respects addition and scalar multiplication — two properties captured by: T(u + v) = T(u) + T(v) and T(cu) = cT(u). Every linear transformation can be represented as matrix multiplication: T(v) = Av for some matrix A.

Linear transformations are the verbs of linear algebra — they describe how space deforms. Rotations, reflections, projections, and shears are all linear. The kernel (null space) contains vectors that get mapped to zero — it measures what the transformation "loses." The image (range) is everything the transformation can produce.

The Rank-Nullity Theorem is one of the most elegant results in linear algebra: rank(T) + nullity(T) = n (the dimension of the domain). It tells us that every dimension the transformation "loses" must be balanced by a corresponding reduction in its image.

  • Linearity Conditions T(u + v) = T(u) + T(v) — additivity. T(cu) = cT(u) — homogeneity. Both must hold for T to be linear. Together they imply T(0) = 0 — a linear transformation always maps the zero vector to the zero vector.
  • Standard Matrix of T Every linear transformation T: ℝⁿ → ℝᵐ has a unique m×n matrix A such that T(v) = Av. The columns of A are the images of the standard basis vectors: A = [T(e₁) | T(e₂) | … | T(eₙ)]. This connects transformations to matrices completely.
  • Kernel (Null Space) ker(T) = {v ∈ ℝⁿ | T(v) = 0}. All vectors mapped to the zero vector. The dimension of ker(T) is called the nullity. A transformation is injective (one-to-one) if and only if its kernel is {0}.
  • Image (Range) im(T) = {T(v) | v ∈ ℝⁿ}. The set of all possible outputs of T. Its dimension is the rank. A transformation is surjective (onto) if and only if its image equals the entire codomain ℝᵐ.
  • Rank-Nullity Theorem rank(T) + nullity(T) = n (where n = dim of domain). This fundamental theorem constrains the structure of any linear map — gain in the image comes at the cost of losing dimensions in the null space, and vice versa.
  • Composition of Transformations If T₁: ℝⁿ → ℝᵐ and T₂: ℝᵐ → ℝᵖ, then their composition T₂∘T₁: ℝⁿ → ℝᵖ is also linear and is represented by the matrix product A₂A₁. The order matters — T₁ is applied first, then T₂.
Rotation by θ
[[cos θ, −sin θ],
[sin θ, cos θ]]
Reflection (x-axis)
[[1, 0],
[0, −1]]
Projection onto x-axis
[[1, 0],
[0, 0]]
Shear (horizontal)
[[1, k],
[0, 1]]
Example — Rotation Transform & Rank-Nullity
// Rotate vector v = (1, 0)ᵀ by θ = 90°
R(90°) = [[cos90°, −sin90°],   [[0, −1],
          [sin90°,  cos90°]] =   [1,  0]]

T(v)  =  [[0, −1],   ×   (1, 0)ᵀ
          [1,  0]]

      =  (0×1 + (−1)×0,  1×1 + 0×0)ᵀ

      =  (0, 1)ᵀ   ← (1,0) rotated 90° → (0,1) ✓

// Rank-Nullity check for R(90°)
rank    = 2   (full rank — rotation preserves all dimensions)
nullity = 0   (only zero vector maps to zero)
rank + nullity = 2 = n  ✓

// Rank-Nullity for projection onto x-axis
P = [[1,0],[0,0]]
rank    = 1   (image = x-axis, 1D)
nullity = 1   (ker = y-axis, all (0,y) → 0)
rank + nullity = 2 = n  ✓
